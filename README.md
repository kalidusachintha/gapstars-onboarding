## Install

- **Clone the repository with git clone**
- **Copy .env.example file to .env**
- **Run composer install**
- **Run php artisan key:generate**
- **Run npm install**
- **Run npm run dev**
- **Launch the app in php artisan serve (http://127.0.0.1:8000)**
